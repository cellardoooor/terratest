# Terraform Yandex Cloud Test Project

Пример инфраструктуры в Yandex Cloud, полностью разворачиваемой через Terraform.  
Проект демонстрирует:

- Создание VPC с подсетью
- Два виртуальных сервера (Compute Instances)
- Сетевой балансировщик (Network Load Balancer)
- NAT для выхода серверов в интернет
